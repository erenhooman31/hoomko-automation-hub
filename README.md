# هاب اتوماسیون کسب و کار

یک نمونه کار فارسی برای نمایش طراحی و توسعه داشبورد پایش Workflow، Webhook و اتصال سرویس ها با رویکرد n8n و API Integration.

## قابلیت ها

- نمایش Workflowهای فعال و سلامت اجرای آن ها
- شبیه سازی مسیر اتصال Webhook، اعتبارسنجی داده و سرویس مقصد
- Run log برای آخرین اجرای آزمایشی
- UI فارسی، RTL و واکنش گرا
- پیاده سازی با React و Vite

## هدف نمونه کار

این پروژه نشان می دهد چگونه می توان فرآیندهای تکراری کسب و کار مثل ثبت لید، ارسال پیامک، گزارش روزانه، اتصال CRM و همگام سازی سفارش ها را در یک مرکز کنترل قابل پایش نمایش داد.

## اجرای محلی

```bash
npm install
npm run dev
```
